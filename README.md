# AI Learning Project

This project contains examples and tutorials for working with Large Language Models (LLMs) and agentic workflows.

## Project Structure

### 1-introduction/
This folder contains the fundamentals and basics on different ways to retrieve data from LLMs:
- **1-basic.ipynb** - Basic concepts and getting started
- **2-structured.ipynb** - Structured data retrieval from LLMs
- **3-tools.ipynb** - Using tools and function calling with LLMs
- **4-retrieval.ipynb** - Data retrieval from pre-defined knowledge base

### 2-workflow-pattern/
This folder explores different agentic workflow patterns for building intelligent systems:
- **1-prompt-chaining.ipynb** - Sequential prompt chaining patterns
- **2-routing.ipynb** - Dynamic routing based on input analysis
- **3-parallelization.ipynb** - Parallel execution of multiple agents
- **4-orchestrator.ipynb** - Orchestrating multiple agents with a coordinator
